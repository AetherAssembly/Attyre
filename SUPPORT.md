# Getting Help

## Check the app first

The version you're running and basic app info are in **Settings → About**. If something looks broken, try the export/import cycle (Settings → Backup & Restore) to rule out a corrupted data issue.

## For bugs and feature requests

Open an issue on the [GitHub issue tracker](https://github.com/AetherAssembly/Attyre/issues). Use the bug report template and include your platform, version, and steps to reproduce.

## For questions and general help

Please don't open a GitHub issue just to ask a question — it creates noise for everyone. Instead:

- **Email:** [support@aetherassembly.org](mailto:support@aetherassembly.org)
- **Contact form:** [forms.gle/T4i7GGzaT3HUrffm9](https://forms.gle/T4i7GGzaT3HUrffm9)

## For security issues

Please do not report security vulnerabilities publicly. See [SECURITY.md](.github/SECURITY.md) for how to report them privately.
